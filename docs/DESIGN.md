# 목표

게임을 개발할 때 따라야하는 UI 디자인 지침을 정의합니다. 게임 간 Look & Feel이 유사할 수 있도록 UI 개발 시 이 지침을 따라야 합니다.

# 지침

## 로비 화면 (LobbyPanel)

### 레이아웃 구조 — 게임 간 통일

모든 게임의 호스트 로비는 공통 `<game-lobby>` 웹 컴포넌트를 사용한다. 레이아웃 구조와 동작은 게임과 무관하게 동일해야 한다.

| 항목 | 규칙 |
|---|---|
| 레이아웃 | QR + 사이드 슬롯 / 참가자 목록 / 시작 버튼 — 동일 |
| 폰트 크기·간격 | LobbyPanel 기본값 유지 |
| 버튼 동작 | 최소 인원 + 전원 준비 시 활성화 |
| 스크롤바 | 없어야 함 (FHD 기준 90vh 내 완전 표시) |

### 게임 테마 색상 — 게임별 고유

강조색과 테두리색은 게임의 시각적 아이덴티티를 반영하므로 게임마다 달라야 한다. `body`에 아래 CSS 변수를 오버라이드하여 적용한다.

```css
body.my-game {
  --lobby-panel-bg:   /* 패널 배경 */;
  --lobby-panel2-bg:  /* 카드 배경 */;
  --lobby-border:     /* 테두리 색 */;
  --lobby-accent:     /* 강조색 (버튼, 하이라이트) */;
  --lobby-accent-dim: /* 강조색 반투명 */;
  --lobby-text:       /* 기본 텍스트 */;
  --lobby-sub:        /* 보조 텍스트 */;
  --lobby-avatar-size: /* 아바타 크기 */;
}
```

| 게임 | 강조색 | 성격 |
|---|---|---|
| 눈치 10단 | 골드 `#f59e0b` | 보드게임, 심리전 |
| Spin Battle | 시안 `#2ce8ff` | 사이버펑크, SF |

> 레이아웃 구조를 통일하면서도 색상으로 게임 간 분위기를 구분한다.
> 강조색을 통일하면 게임 고유의 무드가 사라지므로 통일하지 않는다.

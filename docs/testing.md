# 테스트 가이드

## 실행
```bash
npm install
npm run dev
```
- 서버: `:3000`
- Vite: `:5173`

## 기본 점검
1. 로비에서 멀티/솔로 탭과 게임 카드 표시 확인
2. 게임 진입 후 세션 생성/QR 표시 확인
3. 모바일 입장/준비/게임 시작/리셋 확인
4. 플레이어 재연결(브라우저 새로고침) 확인

## P2P 점검
- 같은 LAN에서 플레이 연결 후 콘솔 로그 확인
- 기대 로그: `[P2P] ... 직접 연결됨`
- 실패 시 동작: Socket.IO 폴백으로 게임 진행 가능

## 프로덕션 점검
```bash
npm run build
node server/index.js
```
`http://<server-ip>:3000` 접속 후 동일 시나리오 테스트

# 그림 릴레이

- 경로: `games/relay-drawing/`
- 인원: 2~8명

## 개요
플레이어가 그림과 단어를 번갈아 제출하며 체인을 완성하는 파티 게임.
라운드 종료 후 결과를 세로 스토리보드로 순차 공개합니다.

## 구현 특징
- 턴별 과제 자동 할당(`roundAssignments`)
- 그림/단어 번갈아 제출
- 제한시간 종료 시 자동 제출(더미 데이터 포함)
- 결과 발표 단계에서 체인 순차 공개
- 모바일 리액션 이모지 송신 지원
- 맞춤형 프로필 아바타(모바일 직접 드로잉)
- 단계별 오디오 시스템(`AudioManager`)

## 시각/UX 개선
- 호스트: 글래스모피즘 기반 UI
- 모바일: 가독성 중심 타이포/헤더 개선
- 단계 전환 애니메이션 강화

## 검증 결과
- 로비 닉네임/아바타 동기화 확인
- 그리기 -> 단어 -> 그리기 턴 순환 확인
- 단계별 BGM/SFX 트리거 확인
- 결과 순차 공개/자동 스크롤 확인

## 최근 버그 수정
- 결과 화면 중복 표시 이슈 수정(이벤트 리스너 중복/제출 가드 보완)
- 라운드당 1회 제출 보장 및 발표 상태 중앙 관리로 안정성 개선

## 주요 메시지
- M->H: `setProfile`, `submitTurn`, `sendReaction`
- H->M: `playerListUpdated`, `gameStarting`, `roundAssignments`, `showResults`

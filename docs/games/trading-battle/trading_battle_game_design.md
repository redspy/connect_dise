
# 차트 트레이딩 배틀 게임 기획서

## 1. 개요

본 게임은 **실제 과거 주식 데이터를 기반으로 한 멀티플레이 트레이딩 경쟁 게임**이다.  
플레이어들은 동일한 차트를 보며 모바일에서 매매 결정을 내리고, 제한된 시간 동안 **누가 더 높은 수익률을 기록하는지** 경쟁한다.

게임은 **공용 화면(대형 디스플레이)**과 **개인 화면(모바일 기기)**의 두 가지 인터페이스로 구성된다.

- 공용 화면: 차트, 시장 정보, 플레이어 순위 등 **게임 진행 정보 공유**
- 개인 화면: 매매 버튼 및 개인 상태 확인 등 **플레이어 조작 인터페이스**

본 구조는 여러 명의 플레이어가 동시에 하나의 시장을 바라보며 경쟁하는 **멀티플레이 트레이딩 배틀 경험**을 제공한다.

---

# 2. 시스템 구조

## 2.1 공용 화면 (TV / 모니터 / 태블릿 등 대형 디스플레이)

공용 화면은 모든 플레이어가 함께 보는 **게임 진행 보드** 역할을 한다.

표시 정보:

- 실제 주식 차트 (캔들)
- 거래량
- 기술적 지표
  - 이동평균선 (5 / 10 / 20 / 60 / 100 / 200)
  - RSI
  - MACD
  - Bollinger Band
- 현재 가격
- 남은 게임 시간
- 남은 캔들 수
- 차트 위 플레이어 매수/매도 지점 표시
- 실시간 플레이어 순위
- 현재 1위 플레이어 강조 표시

공용 화면의 목적:

- 차트 분석
- 시장 흐름 공유
- 경쟁 상황 시각화
- 관전 재미 제공

---

## 2.2 개인 화면 (모바일)

모바일 화면은 **플레이어의 주문 입력 인터페이스**이다.

분석 정보는 최소화하고 **빠른 매매 입력에 최적화**한다.

표시 정보:

상단
- 현재 포지션
- 현재 자산
- 현재 수익률
- 남은 주문 가능 시간

중앙
- 대형 주문 버튼

주문 버튼:

- 2배 롱
- 롱
- 매도 (청산)
- 숏
- 2배 숏

버튼은 큰 크기로 배치하여 **빠른 입력이 가능하도록 설계**한다.

---

# 3. 게임 단계

게임은 다음 단계로 진행된다.

1. 게임 설정 단계
2. 플레이어 참가 단계
3. 게임 시작 및 차트 공개 단계
4. 트레이딩 진행 단계
5. 게임 종료 및 결과 발표 단계

각 단계에서 공용 화면과 개인 화면의 역할이 명확히 구분된다.

---

# 4. 단계별 화면 구성

## 4.1 게임 설정 단계

### 공용 화면

호스트가 게임 조건을 설정한다.

설정 항목:

- 게임 시간  
  - 5분  
  - 10분

- 캔들 공개 간격  
  - 3초  
  - 5초

- 데이터 단위  
  - 일봉

- 기술 지표 표시 여부
  - 이동평균선
  - RSI
  - MACD
  - Bollinger Band

- 레버리지 사용 여부

설정 완료 후 **게임 세션 QR 코드 표시**

### 개인 화면

플레이어는 QR 코드를 통해 접속

표시 정보:

- 닉네임 입력
- 아바타 선택
- 게임 참가 대기 화면

---

## 4.2 플레이어 참가 단계

### 공용 화면

- 참가 플레이어 목록 표시
- 최대 4명 참가
- 참가 완료 시 게임 시작 버튼 활성화

### 개인 화면

- 대기 화면
- 플레이어 준비 상태 표시

---

## 4.3 게임 시작 단계

### 공용 화면

- 과거 차트 히스토리 표시 (약 200~250봉)
- 게임 시작 카운트다운

표시 요소:

- 차트
- 거래량
- 기술적 지표
- 플레이어 패널

### 개인 화면

초기 상태 표시

- 현재 자산
- 현재 포지션 (현금 상태)
- 주문 버튼 활성화

---

## 4.4 트레이딩 진행 단계

차트는 일정 시간 간격으로 **새로운 캔들이 공개**된다.

예:
- 3초마다 1캔들

### 공용 화면

표시 정보:

- 실시간 차트 업데이트
- 플레이어 매수/매도 지점 표시
- 플레이어 순위
- 현재 1위 플레이어 강조 표시

하단 플레이어 패널:

각 패널 표시

- 플레이어 이름
- 현재 자산
- 현재 수익률
- 현재 포지션 상태

포지션 상태 아이콘:

- 롱
- 숏
- 2배 롱
- 2배 숏
- 현금 상태

### 개인 화면

플레이어는 버튼을 눌러 매매

버튼:

- 2배 롱
- 롱
- 매도
- 숏
- 2배 숏

주문은 다음 캔들 공개 시점에 체결된다.

---

## 4.5 게임 종료 단계

게임 시간이 종료되면 차트 공개가 멈춘다.

### 공용 화면

표시 내용:

- 최종 차트 전체 공개
- 플레이어 매매 지점 표시
- 최종 순위

순위 기준:

최종 자산

1위 플레이어 강조 표시

### 개인 화면

플레이어 개인 결과 표시

- 최종 자산
- 총 수익률
- 최종 순위

---

# 5. 게임의 핵심 재미 요소

- 실제 주식 데이터를 활용한 현실감 있는 차트
- 기술적 분석 기반 경쟁
- 실시간 수익률 순위 경쟁
- 플레이어 매매 타이밍 공개로 인한 심리전
- 레버리지 선택에 따른 높은 리스크/보상

---

# 6. 요약

본 게임은

**"실제 주식 차트를 함께 보며 모바일로 매매를 실행하고, 제한된 시간 동안 최고의 수익률을 겨루는 멀티플레이 트레이딩 경쟁 게임"**이다.

공용 화면과 개인 화면의 역할을 명확히 분리하여
- 공용 화면은 **시장과 경쟁 상황을 보여주고**
- 개인 화면은 **빠른 매매 입력을 담당한다**.

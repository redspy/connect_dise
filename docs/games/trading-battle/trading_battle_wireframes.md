# 차트 트레이딩 배틀 게임 와이어프레임 UI

## 1. 문서 개요

본 문서는 **차트 트레이딩 배틀 게임**의 화면 구성을 단계별로 정리한 와이어프레임 문서이다.  
게임은 **공용 화면(대형 디스플레이)**과 **개인 화면(모바일)**으로 구성되며, 각 단계마다 두 화면의 역할이 명확하게 구분된다.

---

# 2. 화면 구성 원칙

## 2.1 공용 화면

- 차트와 경쟁 상황을 모두가 함께 본다.
- 플레이어 간 순위 경쟁과 포지션 상태를 직관적으로 보여준다.
- 차트, 현재 1위, 플레이어 패널이 핵심 요소다.

## 2.2 개인 화면

- 복잡한 정보보다 **빠른 주문 입력**에 집중한다.
- 버튼은 크고 단순하게 배치한다.
- 현재 포지션, 자산, 수익률만 최소한으로 표시한다.

# 4. 공용 화면 핵심 UI 요소 정리

## 4.1 상단

- 현재 1위 플레이어 강조
- 현재 1위 수익률
- 남은 시간

## 4.2 중앙

- 메인 캔들 차트
- 플레이어 매수/매도 지점 실시간 표시
- 거래량
- 보조지표

## 4.3 하단

- 4명의 플레이어 패널
- 이름
- 자산
- 수익률
- 포지션 상태 아이콘

---

# 5. 개인 화면 핵심 UI 요소 정리

## 5.1 상단

- 현재 포지션
- 현재 자산
- 현재 수익률
- 주문 가능 시간

## 5.2 중앙

- 주문 버튼 5개
  - 2배 롱
  - 롱
  - 매도(청산)
  - 숏
  - 2배 숏

## 5.3 설계 원칙

- 한 손으로 빠르게 누를 수 있어야 함
- 아이콘 중심으로 즉시 구분 가능해야 함
- 버튼이 크고 단순해야 함

---

# 6. 요약

본 와이어프레임은 다음 방향을 전제로 한다.

- 공용 화면은 **차트 분석 + 순위 경쟁 + 관전 재미**
- 개인 화면은 **즉시 주문을 위한 입력 장치**
- 단계별로 두 화면의 역할을 분리하여 **게임의 몰입감과 긴장감**을 높인다.

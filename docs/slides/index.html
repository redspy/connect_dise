03 / MANAGEMENT & COOPERATION

🤝 PL 협업 조율 및 3대 핵심 의사결정

에이전트 조율을 통한 고성능 아키텍처 및 품질 타협점 수립

PL은 기획 의도와 기술적 안정성 간의 가교 역할을 수행하며, 의사결정 일지(docs/decision-log.md)에 아래 3가지 원칙을 수립/기록했습니다.

아키텍처 기밀 유니캐스트

스파이 정체와 단어는 broadcast를 일절 차단하고 sendToPlayer로만 송신하여 브라우저 개발자 도구 패킷 스니핑에 의한 스파이 부정행위를 사전 격퇴.

인터랙션 햅틱 햅틱 피드백

모바일 폰의 터치감 보강을 위해 3D Flip Card 개방, 지목 투표 클릭, 최종 승패 브리핑 판정 시 모션 진동(Vibration) 피드백 패턴을 개별 적용.

품질 TDD 검증 우선 강제

임의의 소스코드 체크인 방지를 위해, 추첨 알고리즘의 편향성과 투표 다수결 동률 계산 로직을 유닛테스트로 코드 통합 전 먼저 선행 검증하도록 조율.

04 / TECHNICAL DESIGN

⚙️ 개발자 기술 설계 및 Base 클래스 상속

플랫폼 코어와 콘텐츠 레이어의 안전한 분리

플랫폼의 두 핵심 베이스 클래스를 상속받아, 서버 메모리를 거치지 않는 초경량 상태 머신을 완성하였습니다.

  • HostBaseGame 상속 설계: onSetup ➔ onPlayerJoin ➔ onAllReady ➔ onPlayerRejoin ➔ onReset 생명주기 훅을 완벽히 매칭하고, 호스트용 5개 페이즈(lobby, setup, discussion, voting, result) 제어 루틴을 수립.
  • MobileBaseGame 상속 설계: 모바일용 5개 제어화면(setup-profile, waiting, role-reveal, submit-hint, vote) 뷰 포인트를 선언하고 햅틱 피드백 조율.

🔄 세션 복구(Rejoin) 설계

모바일 탭의 순간 이탈이나 새로고침 시 정보 소실을 방지하기 위해 호스트 메모리 내에 배정 상태를 기억하는 Map 장치를 내장했습니다.

복귀 유저 감지 즉시 역할과 키워드를 **개별 복원 패킷**으로 수동 하달하여 유연성을 강화했습니다.

05 / FULLSTACK IMPLEMENTATION

💻 호스트 및 모바일 클라이언트 상세 구현

💬 힌트 단어 플로팅

플레이어들이 모바일로 전송한 단어 힌트를 TV 화면에 물리 엔진 효과와 무작위 주기 오프셋을 준 CSS floating-bubble로 구현. 시각적 활기를 부여.

🃏 3D Flip Card 인터랙션

모바일 폰의 은밀성을 보존하기 위해, 터치 시 부드럽게 180도 회전(Y축 3D Transform)하며 숨겨둔 시민/스파이 비밀 정보를 개방하는 UX 구현 완료.

🚨 지목 투표 UI 및 실시간 취합

자신을 제외한 나머지 생존 플레이어들의 목록을 모바일에 버튼 리스트로 동적 생성. 득표 현황을 TV의 프로그레스 게이지로 실시간 연동 처리.

Vite HMR 핫 리로딩 및 멀티 빌드 컴파일 구성으로, 로컬/사설망 어디서나 무설치 브라우저 터치 기반 조작이 100% 매끄럽게 작동하도록 결합되었습니다.

06 / QUALITY ASSURANCE

🧪 QA 품질 검증 및 엣지 케이스 정복

인수 시나리오 및 예외 사항 전수 통과

품질 에이전트는 기획 인수 조건을 100% 만족하는지 E2E 플레이 검증 및 자동화 테스트를 수행하여 품질 합격(PASS)을 선언하였습니다.

  • Playwright E2E tests/hidden-agent/game.spec.js에 3인 플레이 풀-시나리오 E2E 자동 테스트 코드를 완비하여 지속 가능한 회귀 테스트 기반 마련.
  • Robustness 새로고침(Rejoin) 복원성 검증: 단어 입력창 도중 새로고침 재진입 시 기존 역할과 단어가 정상 하달되어 플레이어가 멈춤 없이 100% 동기화됨을 입증.

🔌 P2P Fallback 투명 폴백 테스트

인위적으로 기기 간 P2P 데이터 채널 터널링을 차단하고 강도 높은 예외 환경을 모의 설계했습니다.

전송 차단 즉시 80ms 이내에 Socket.IO 서버 릴레이 파이프로 자동 백업 이행되어 힌트 전송 및 투표 통신에 어떠한 지연 체감도 주지 않았음을 확인하였습니다.
